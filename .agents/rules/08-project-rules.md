# Repository Intelligence Platform

This project is built incrementally.

Current priorities are:

Repository understanding

Static analysis

Metadata extraction

Repository exploration

Dependency analysis

Semantic search

AI-assisted reasoning

## AI Usage

Always prefer deterministic techniques before AI.

Prefer:

AST parsing

Symbol extraction

Metadata indexing

Graph traversal

Static analysis

Use LLMs only when deterministic methods cannot solve the problem effectively.

AI should enhance repository understanding, not replace it.

## Scope

Implement only the current phase.

Do not implement future phases.

If a feature belongs to a later phase, explain why instead of implementing it.
